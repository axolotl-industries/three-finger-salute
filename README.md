# three-finger-salute
A Mac app for three-finger gestures

As a dyed-in-the-wool Windows user attempting to live on a Mac for a while, I found myself annoyed at the lack of usefulness of three-finger trackpad gestures. So I threw together an app to deal with it.

This app does two things:
1) Three finger swipe up and down to control volume
2) Three finger tap for middle mouse click

That's all.
